## Yocto metadata layer for XLibre support

- This is an _*ALPHA*_ of the support set for Yocto Scarthgap.
- It works with a little bit of massaging of stuff as there is still some things (Like a Login Manager vetted for things)
- It has been _*validated*_ on a Raspberry Pi5 at this time but nothing else in my stuff (I have a lot o' stuff, mind...)

It is a WIP.  It will be growing over time as I intend on working WITH XLibre in the future. It is being done because I see a LOT of push for the still not yet ready for prime time Wayland as being forced into the fore from people EMPLOYED by Red Hat, and at least encouraged by the same.  This is solid compared to it.  It is hoped it can and will be or will be useful to others over time.
