## What's in this directory?

This contains recipes for all the GitHub active maintained drivers for XLibre with the appropriate version, ABI, etc. set.

It is CURRENTLY not complete.  Much of what was in the XOrg stack here in Yocto-land was predicated on things more likely to figure in an Embedded contexr.  Over time, there will be more and more of the proper upstreams moved in for the recipe framework for build.
